Using React.js to Produce a Dashboard UI

February 2022

During February, I started to implement the stock market dashboard UI that I designed as part of December's project.

This is an application I'm passionate about completing to a high standard - as I intend to use it myself whilst I continue to invest in the stock market. As such, I am cutting no corners, ensuring the front-end looks and functions exactly how I want. I'm really happy with the progress I'm making (I'd guess I'm around 60% complete), but to avoid feeling burnt out with this project, I've decided to extend this month's project to cover March as well.

I am on track to complete the UI by the end of March, and at that point I'll post a full description of my work on the front-end
