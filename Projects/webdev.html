Project I: Web Development

September 2021

In 2014, I decided I wanted to study computer science at University. Given that my school didn't offer it as an A-Level, I was worried I'd be behind everyone, so I taught myself HTML, CSS and JavaScript using online tutorials. Since then, I've used HTML and CSS throughout the last 7 years on various personal projects, university modules and on my placement year. So putting together a simple website like this wasn't much of a learning curve.

There were two elements that were new to me, though:

  1. Hosting
    Even though I've made plenty of web pages over the last few years, I've never owned a domain or put too much thought into the back-end infrastructure that goes into web development. After a bit of research, I bought the domain charliebignell.uk and paid for hosting via Pilchard.co.uk. I then developed the webiste using GitHub Pages. I've used Pages a couple of times and it's a really quick and easy way to get a website up and running quickly, and I was already uploading the cde to GitHub so it seemed like the best choice. I then had to tell GitHub Pages to use my cutom domain, which was more complicated than expected but I got there in the end.
